# How to Compile and Run C# Samples with .NET SDK

## Prerequisites

* Download and install the [.NET SDK](https://dotnet.microsoft.com)

## Compile and Run a Sample Project

1. Navigate to the project directory:
   
```bash
cd path/to/sample
```

2. Build the project:

```bash
dotnet build
```

3. Run the project:

```bash
dotnet run
```
